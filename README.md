# API created for "CoronaVac" Project, project made for college

To start the api use
```bash
npm install
```
And

```bash
npm i -g ngrok adonis
```
```bash
npm install sqlite3
```
Then

```bash
adonis migration:run
```

To start server
```bash
adonis serve --dev
```

To start NGROK
```bash
npm run rocknroll
```
